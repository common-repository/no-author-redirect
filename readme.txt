Stop redirects from URLs containing the `author` query parameter and user ID to permalinks with usernames to make it a little harder to iterate through the system and collect valid usernames.

== Description ==

Stop redirects from URLs containing the <code>author</code> query parameter and user ID to permalinks with usernames to make it a little harder to iterate through the system and collect valid usernames.
